# Machine Learning Univariate Time Series Analysis of Two Datasets using Random Forest
## Data Used:
1. Statistics in Modern Technology Dataset
2. Statistical Methods in Artificial Intelligence (AI)
The data was gotten from publication records from Semantic Scholar (https://www.semanticscholar.org)

## Tools Used: Python, Microsoft Excel and Jupyter Notebook (for IDE)
### Python Packages Used
1. pandas
2. numpy
3. matplotlib.pyplot
4. RandomForestRegressor from sklearn.ensemble
5. mean_absolute_error and mean_squared_error from sklearn.metrics
6. plot_acf and plot_pacf from statsmodels.graphics.tsaplots
